# Operator Asignment, Increment, dan Decrement

## 1. Assignment

```angular2html
a = a + 2
```
dapat juga ditulis:
```angular2html
a += 2;
```
dan juga dapat menggunakan tanda lain, seperti:
```angular2html
-=
*=
/=
```

## 2. Increment
### a. Preincrement
```angular2html
++a
```
### b. Postincrement
```angular2html
a++
```

## 3. Decrement
### a. Predecrement
```angular2html
--a
```
### b. Postdecrement
```angular2html
a--
```
